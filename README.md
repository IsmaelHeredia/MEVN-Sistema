# MEVN Sistema

**! Components :**

MongoDB 4.0.5

Express.js 4.17.2

VueJS 3.2.13

Vuetify 3

Node.js 16.13.2

**Commands :**

**! Backend**

```
npm install
node index.js
Load http://localhost:3030/instalar to install tables and default values in the database
```

**! Frontend**

```
npm install
npm run serve
```
